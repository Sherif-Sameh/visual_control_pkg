# Calibration Package

## Package Details

Provides calibration routines required for accurate vision-guided manipulation. The package contains workflows for hand-eye calibration & evaluation, and eye calibration through joint eye multi-view pose, geometry (mesh), and texture optimization using differentiable rendering via NVIDIA Kaolin and TCP-to-camera calibration via PyTorch3D. Calibration routines are designed to integrate directly with the vision-based control & planning pipeline and can be executed independently through dedicated system launch files.

## Primary Responsibilities

* Hand-Eye Calibration & Evaluation
* Eye Calibration
* TCP Calibration
