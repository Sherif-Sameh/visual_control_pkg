# Robot Description Package

## Package Details

Contains robot descriptions and supporting resources used throughout the project. The package is built upon the [`ur_description`](https://github.com/UniversalRobots/Universal_Robots_ROS2_Description) ecosystem and provides the robot, end-effector, and robot cell models required for visualization, planning, kinematics, and simulation.

## Primary Responsibilities

* Robot, End-Effector, and Robot Cell URDF Descriptions
* Mesh and Texture Resources
* Robot configuration files
* Visualization resources
